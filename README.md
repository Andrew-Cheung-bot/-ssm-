#### 软件架构
本项目基于SSM框架，使用WebSocket通信协议与硬件端进行实时连接，前端与后端通过Http协议进行数据交互，后台管理Web端通过Ajax与后台进行异步信息交互，Web端采用Html+Ajax来代替SSM框架中默认使用的Jsp页面。


#### 安装教程

1.  clone本项目到本地
2.  通过Maven构建war包
3.  登陆服务器的tomcat进行部署
